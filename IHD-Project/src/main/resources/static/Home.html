Indian Heritage Database website is developed for people in need who wants to connect with
their culture.This website have tremendous amount of cultural data stored in it.
This website (Indian heritage database) will help in increase their knowledge, verification,
and explore new domain in art and culture. This website has many holy books, Puranas, Upangas
and many other books indexed in it. This website is built on Java Language which uses Apache
Solr search platform to index, highlight and translate. Front end GUI is built on HTML, CSS
and Java Script which gives smooth finishing of website.
